### 기능 Feature
* 보정 피치 조절
  * 최소 0피치 초과 최대 500피치까지 허용되며 소수점도 사용할 수 있습니다.
* 보정 최솟값 조절
  * 보정에서 한 바퀴가 돈 후에 클릭하면 보정되는 값보다 적은 값이 적용되는 문제를 해결할 수 있습니다. 
* 보정 음악 반복
  * 횟수만큼 반복하며 설정한 횟수가 끝나면 보정 메시지가 뜹니다.
* 보정 자세한 표시
  * 보정 중 평균값, 최댓값, 최솟값을 표시합니다.
* 보정 팝업
  * 게임을 플레이하다 죽을 시 보정값을 조절할지 물어보는 팝업을 띄웁니다.
